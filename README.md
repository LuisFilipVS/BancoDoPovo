# Banco Do Povo
Projeto em grupo de uma aplicação WEB simulando um banco financeiro.
Front-End: JavaScript + Vue.js
Back-End: TypeScript

## Project Setup
```sh
npm install
```
### Compile and Hot-Reload for Development
```sh
npm run dev
```
### Compile and Minify for Production
```sh
npm run build
```
